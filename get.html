Memanfaatkan Contact Picker API di Chrome Android untuk memilih kontak secara aman (tanpa perlu membaca seluruh buku telepon).

Cara pakai & catatan kompatibilitas
  • Fitur ini umumnya tersedia di Chrome/Edge/Brave Android dengan HTTPS.
  • Harus dijalankan pada Secure Context (alamat https:// atau localhost).
  • API yang digunakan: navigator.contacts.select(). Properti yang bisa diminta: name, tel, email, address, icon.
  • Privasi: pengguna selalu memilih kontak secara manual, situs tidak dapat mengakses seluruh buku telepon.
